discordサーバーに登録できる絵文字やスタンプのサイズ制限がきついので作ったbot
ユーザー辞書みたいなノリで特定の文字列に画像urlを登録してそれを返信します
urlが返すコンテンツを調べる機能はついていないのでurlであればgifとか動画とか、もうurlであればなんでも実装できちゃいます


ソースフォルダの名前にgyazoが入っていますが機能的にはgyazoは一切関係ないです

gyazoAPIとの連携はよく考えたらする必要がないので諦めました。


python環境インストール
> pythonバージョンは3.10.8
pycordをpip install 
起動方法
discordの開発者向けサイトでbot作成
OSの環境変数にDISCORD_TOKENの名前でトークン追加し再起動
そして起動するだけっす！


現在実装できている機能
> 特定の単語にurlを登録し、bot以外のユーザーがメッセージを送信する都度内容をチェック。登録済みの単語をキーとして値であるurlでbotが返信を行う。
