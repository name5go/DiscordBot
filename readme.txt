趣味で作成しているDiscordBotです。
-----
動作環境
python3.10.8
使用ライブラリ
pycord
openai

-----
現在実装できている機能

◆ChatGPTにずんだもんっぽいロールプレイの設定を渡して、それとのチャットでやり取りできる機能
◆チャットでやり取りしているユーザーがVCにインしているのなら、Botがそのチャンネルに自動で入室しChatGPTからの返答をVoiceVoxEngineに渡し、ずんだもんに読み上げさせることができる機能。
◆スラッシュコマンドによる、ユーザー辞書みたいな感じで、登録された特定のキーワードに反応してその辞書配列上の値である画像URLで直接リプライを行う。(Discordの機能に縛られないカスタムスタンプのようなものを作りたかった)
◆コマンドからEmbedを作成、プルダウンメニューから送信先のテキストチャンネルを選択してそこに投稿する機能。
◆特定のVCチャンネルのユーザー増減に反応して動的に聞き専ミュートテキストチャンネル、ボイスチャンネル、それらを含んだカテゴリーを作成したり削除する機能(Laz鯖で使われているBotに聞き専用のテキストチャンネル作成機能もつけたかった)

-----
今後実装したい機能

◆OpenAi、VoiceVoxEngineとのやり取りにおけるrequest処理周りを非同期処理に変更する。
◆youtube niconico動画の動画URLを渡して音楽を再生させる機能
◆ChatGPTを利用した、TRPGのようなプルダウンメニューから選択肢を選び進める、テキストチャンネル上で遊ぶことのできるアドベンチャーゲームのようなコンテンツの作成
◆リアクションロール
◆GitHub上のリポジトリの変化を通知させる機能(SlackのGitHubApp)のようなものを作りたい。
◆JavaScriptでウェブサイト上からBot管理を行うGUIの作成
